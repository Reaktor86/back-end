композер нужен
1. для внедрения в проект библиотек других разработчиков
2. автозагрузка классов

spl_autoload - в php есть функция для загрузки классов, но с ней сложно
composer упрощает всё

создать папку project

$ composer install - установить зависимости и автолоад

в папке проекта composer init - инициализация самого проекта, чтобы создался composer.json

composer.json - конфигурационный файл с зависимостями и автозагрузкой

PSR-0 стандарт автозагрузки - его код надо добавить в composer.json
накчинается с autoload
только вместо psr-0 написать psr-4
указываем namespace и папку где хранится:
"Base\\": "Base"
где вторая Base - это папка в папке Project, там он будет искать классы

создаем класс BaseClass.php
в файле BaseClass.php вставляем:
namespace Base

в проекте $ composer install

если создался vendor значит все ок

в основном файле project/run.php написать
require 'vendor/autoload.php';
$obj = new \Base\BaseClass();
var_dump($obj);

как подключить уже существующий проект?
в папке проекта composer init

при любом изменении в конфиге надо делать инсталл заново
